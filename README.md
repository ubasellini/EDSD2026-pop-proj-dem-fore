# EDSD 2025/2026: Course on Population Projection & Demographic Forecasting

This repository contains the materials for the course on "Population Projection & Demographic Forecasting", prepared for the 2025/2026 cohort of the [European Doctoral School of Demography](https://eaps.nl/edsd). 

Start date: 27/04/2026

End date: 30/04/2026

## Course overview

### Day 1
- Introduction to population projections
- The cohort component method: projection of a two-sex closed population

### Day 2
- Matrix projections: two-sex closed population

### Day 3
- Extensions of matrix projections: 
    - open population 
    - time-varying assumptions
    
### Day 4
- Introduction to demographic forecasting
- The Lee-Carter model
